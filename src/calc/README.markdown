Calc is an implementation of a simple arithmetical calculator with
integer constants and the operations
* addition `+`
* subtraction `-`
* negation `-`
* multiplication `*`
* division `/`

Example interaction:

    calc -- programming languages zoo
    Type Ctrl-D to exit
    calc> 2+2
    4
    calc> 101 * 111
    11211
    calc> 24 / (4 - 4)
    Error: division by zero
